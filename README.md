# NestJS + MongoDB: Pokedex

1. Clone repository
2. Have NestCLI installed

```bash
npm install -g @nestjs/cli
```

3. Run

```bash
yarn install
```

4. Run MongoDB

```bash
docker-compose up -d
```

## Tech Stack

- NestJS
- MongoDB
